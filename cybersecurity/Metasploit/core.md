# Metasploit learning

Main command

```bash
msfconsole
```

Loading modules

```bash
use <module_name>
```

Setting arguments when inside loaded module

See Modules options

```bash
show options
```
Set options

```bash
set <OPTION> <VALUE>
```

Unset options

```bash
unset <OPTION>
```

Run the module 

```bash
run
```
